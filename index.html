Get Started in Minutes

1

Download Termux

Get Termux from F-Droid or Google Play Store and install it on your Android device.

2

Launch the App

Open Termux and you'll see a terminal prompt ready for commands.

3

Update Packages

Run apt update && apt upgrade to ensure all packages are up to date.

4

Start Exploring

Install tools, write scripts, and explore the full power of Linux on your Android device.

# Update system packages apt update && apt upgrade -y # Install your favorite tools apt install python node-js git curl # Start coding! python --version
