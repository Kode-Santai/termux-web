üîß Installation & Setup

System Requirements

  • Android Version: Android 5.0 or higher
  • Storage: Minimum 100MB free space
  • RAM: At least 1GB recommended

Installation Steps

  1. Download Termux: Get it from F-Droid (recommended) or Google Play Store
  2. Install the App: Open the installer and tap "Install"
  3. Launch Termux: Tap the Termux icon to open the terminal
  4. Grant Permissions: Allow access to storage if prompted

Initial Setup

After first launch, update your system:

$ apt update && apt upgrade -y
üí° Tip: F-Droid is recommended because it offers regular updates and doesn't have Play Store restrictions.

üì¶ Package Management

Using apt (Package Manager)

Termux uses apt for package management, similar to Debian/Ubuntu Linux.

# Update package list $ apt update # Upgrade all packages $ apt upgrade # Install a package $ apt install git # Remove a package $ apt remove git # Search for packages $ apt search python # Clean up $ apt clean

Popular Packages to Install

# Essential tools $ apt install git curl wget nano vim # Python development $ apt install python python-pip # Node.js and npm $ apt install nodejs npm # Ruby $ apt install ruby # Java $ apt install openjdk-11 # Build tools $ apt install build-essential clang
‚ö†Ô∏è Important: Always run apt update before installing new packages to get the latest package information.

üíª Development Tools

Python Development

# Install Python $ apt install python python-pip # Check version $ python --version # Install packages with pip $ pip install requests flask django # Create virtual environment $ python -m venv myenv $ source myenv/bin/activate

Node.js Development

# Install Node.js $ apt install nodejs npm # Check versions $ node --version $ npm --version # Create new project $ mkdir my-project && cd my-project $ npm init -y # Install dependencies $ npm install express

Git & Version Control

# Install Git $ apt install git # Configure Git $ git config --global user.name "Your Name" $ git config --global user.email "email@example.com" # Clone a repository $ git clone https://github.com/username/repo.git # Basic Git workflow $ git add . $ git commit -m "Your message" $ git push origin main

üåê Networking & Web Servers

HTTP Tools

# Download files with curl $ curl -O https://example.com/file.zip # Download with wget $ wget https://example.com/file.zip # Check network connectivity $ ping google.com # Show network interfaces $ ifconfig

Python Web Server

# Python 3 simple HTTP server $ python -m http.server 8000 # Access at: http://localhost:8000 # or use your device IP address

Node.js Web Server

# Create server.js $ nano server.js # Content: const http = require('http'); const server = http.createServer((req, res) => { res.writeHead(200); res.end('Hello from Termux!'); }); server.listen(3000); # Run server $ node server.js

Install Nginx

# Install Nginx $ apt install nginx # Start Nginx $ nginx # Access at: http://localhost:8080 # Stop Nginx $ nginx -s stop

ü§ñ Scripting & Automation

Bash Scripting

# Create a script file $ nano myscript.sh # Example script: #!/bin/bash echo "Hello, Termux!" echo "Current date: $(date)" ls -la # Make it executable $ chmod +x myscript.sh # Run the script $ ./myscript.sh

Python Automation

# Install required packages $ pip install requests beautifulsoup4 # Create automation script $ nano bot.py # Example script: import requests from datetime import datetime response = requests.get('https://api.github.com') print(f"GitHub API Status: {response.status_code}") print(f"Current Time: {datetime.now()}") # Run the script $ python bot.py

Scheduled Tasks with Cron

# Edit crontab $ crontab -e # Example cron entries: # Run daily at 9 AM 0 9 * * * /path/to/script.sh # Run every hour 0 * * * * python /path/to/bot.py # Run every 30 minutes */30 * * * * bash /path/to/backup.sh

‚öôÔ∏è Advanced Configuration

Customizing Shell

# Install zsh $ apt install zsh # Change default shell to zsh $ chsh -s zsh # Install Oh My Zsh (optional) $ sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Environment Variables

# Edit profile $ nano ~/.bashrc # Add custom variables export MY_VAR="value" export PATH=$PATH:/path/to/bin # Apply changes $ source ~/.bashrc

SSH Setup

# Install OpenSSH $ apt install openssh # Generate SSH keys $ ssh-keygen -t rsa -b 4096 # Start SSH server $ sshd # Connect to remote server $ ssh user@example.com

Storage Access

# Request storage permission $ termux-setup-storage # Access shared storage $ cd ~/storage/downloads $ ls
üîê Security Note: Keep your Termux environment secure by:
  • Regularly updating packages with apt update && apt upgrade
  • Using strong passwords if setting up SSH
  • Avoiding running untrusted scripts
  • Keeping sensitive files private with correct permissions
